# Pirate Rose Webroot

This is the homepage for the Pirate Rose Commune!

## Folders

- **assets** contains a bit of CSS, fonts, and images.
- **\_data** contains information about links and where they should go.
- **\_includes** contains header and footer info.
- **\_sass** contains the main styling files for the page.
- **\_layouts** contains the templates for the page.
    
## Main Files
- **announcements.md** has announcements info.
- **happenings.md** contains the happenings info.
- **us.md, you.md, more.md** contain the content for those pages.
- **\_sass/main.css** controls the main styling for the page.
